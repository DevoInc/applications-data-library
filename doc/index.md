# Documentation

The main objective of this library is to provide a standard way to make requests for data so they can be then adapted to the widget that consumes them.

There are several classes that allow you to make requests with different purposes. The classes are as follows:

## Request

- [Request](request/Request.md)
- [RequestAPI](request/RequestAPI.md)
- [RequestChain](request/RequestChain.md)
- [RequestApiMock](request/RequestApiMock.md)
- [RequestCustom](request/RequestApiCustom.md)
